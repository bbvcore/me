Javascript es el principal y más usado lenguaje de programación del lado cliente, además esta en constante evolución y permite realizar aplicaciones completas como lenguaje base, tanto en el backend como en el frontend. Además, también posee una amplia gama de librerías y frameworks para permitir gestionar mejor la modularidad y escalabilidad.

Php es el principal lenguaje de programación de lado del servidor en la web, hay otras alternativas como puede ser usar Java, C#, python o incluso Javascript a través del uso de NodeJS, aún así sigue siendo el lenguaje de backend con más penetración en la red de redes y parte fundamental de la misma desde los 90. Además ha evolucionado adaptándose a los nuevos tiempo y también se han desarrollado sistemas de plantillas, CMS o Frameworks que permiten expandir su influencia en la red y en las aplicaciones.

Wordpress se trata de un CMS, sistema de gestión de contenidos, construido con base en Php y que goza de una envidiable difusión. Este CMS se utilizaba inicialmente como blog, pero es mucho más, permite construir aplicaciones web complejas además de ofrecer la posibilidad crear con facilidad mediante el uso de Woocommerce plataformas de comercio online. Entre los principales usuarios de este CMS a nivel global destacan varios como son BBC America, la universidad de Harvard, Sony music, Disney (tienda online), Mercedes o Microsoft News.
